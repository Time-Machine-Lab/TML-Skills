## 💡 What is TML-Skills?

**TML-Skills** (Trend, Magic, Logic) is a curated collection of high-quality, production-ready **AI Skills** designed for Trae IDE.

Unlike generic prompts, these skills are engineered with **industry best practices**, **domain knowledge**, and **viral logic**. They turn your AI assistant from a junior coder into a domain expert.

> "Don't just prompt. **Skill up.**"

## ✨ Features

- **🧠 Domain Expert Logic**: Each skill is backed by real-world research and expert workflows.
- **🔌 Plug & Play**: Drop them into your `.trae/skills` folder and start using them instantly.
- **📈 Growth Hacking**: Built-in logic for viral content creation (e.g., Xiaohongshu, TikTok).
- **🎨 Aesthetic First**: We prioritize output quality, structure, and visual appeal.

## 🚀 Quick Start

Get these skills into your Trae IDE in seconds.

### Option A: The "Pro" Way (Git Clone)
Recommended for easy updates.

```bash
# 1. Go to your Trae skills directory
cd .trae/skills

# 2. Clone this repository
git clone https://github.com/Time-Machine-Lab/TML-Skills.git
```

### Option B: The "Lazy" Way (npx degit)
Download without git history (requires Node.js).

```bash
# Run this in your project root
npx degit Time-Machine-Lab/TML-Skills .trae/skills/TML-Skills
```

### Option C: Manual Install
1. Download the `.zip` of this repo.
2. Extract the folders inside `.trae/skills/` to your project's `.trae/skills/` directory.
3. Reload Trae.

## 🕹️ Usage

Once installed, simply open a chat in Trae and type:

> "I want to create a Xiaohongshu post about **Cyberpunk City**."

The AI will automatically detect the intent and invoke the `xiaohongshu-niche-wallpaper` skill to assist you.
