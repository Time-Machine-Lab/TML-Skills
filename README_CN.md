## 💡 什么是 TML-Skills?

**TML-Skills** (Trend, Magic, Logic) 是一个专为 Trae IDE 打造的高质量、生产级 **AI Skills** 精选集。

不同于普通的提示词（Prompts），这里的每一个 Skill 都经过精心设计，融合了**行业最佳实践**、**领域专业知识**和**病毒式传播逻辑**。它们能瞬间将你的 AI 助手从初级程序员变身为领域专家。

> "不要只是提问，**让 AI 掌握技能。**"

## ✨ 核心特性

- **🧠 领域专家逻辑**：每个 Skill 背后都有真实的业务研究和专家工作流支撑。
- **🔌 即插即用**：只需放入你的 `.trae/skills` 文件夹，即可立即使用。
- **📈 增长黑客**：内置病毒式内容创作逻辑（如小红书、抖音等平台的爆款公式）。
- **🎨 审美优先**：我们极度重视输出内容的质量、结构和视觉美感。

## 🚀 快速开始

只需几秒钟，即可将这些 Skills 引入你的 Trae IDE。

### 方案 A：极客首选 (Git Clone)
推荐使用，方便后续一键更新。

```bash
# 1. 进入你的 Trae skills 目录
cd .trae/skills

# 2. 克隆本仓库
git clone https://github.com/Time-Machine-Lab/TML-Skills.git
```

### 方案 B：懒人模式 (npx degit)
无需 Git 历史，直接下载文件（需要 Node.js 环境）。

```bash
# 在你的项目根目录下运行
npx degit Time-Machine-Lab/TML-Skills .trae/skills/TML-Skills
```

### 方案 C：手动安装
1. 下载本仓库的 `.zip` 压缩包。
2. 将解压后的文件夹复制到你的项目 `.trae/skills/` 目录下。
3. 重启 Trae 或重载窗口。

## 🕹️ 使用指南

安装完成后，直接在 Trae 的聊天窗口中输入：

> “我想发一篇关于 **赛博朋克城市** 的小红书。”

AI 会自动识别意图，并调用 `xiaohongshu-niche-wallpaper` skill 来辅助你创作。
