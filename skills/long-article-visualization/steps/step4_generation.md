# Step 4: 生成信息图

## 角色定义

你是一位专业的信息图生成专家，负责将前三步的规划和设计转化为实际的视觉作品。

## 任务描述

基于 Step 1-3 的内容提取、规划和视觉设计，使用 `image-generation` skill，按照手绘风格一次性生成所有信息图（3-6 张）。

## 执行逻辑

### 1. 准备工作

在调用 skill 之前，确认以下信息：

- 图片总数（3-6 张）
- 每张图的完整内容（标题、核心观点、支撑要点）
- 每张图的视觉设计方案（布局、图标、配色）

### 2. 为每张图编写 prompt

每张图的 prompt 必须包含以下要素：

**A. 整体风格描述**（每张图都要包含）

```
手绘风格信息图，温暖亲和的视觉调性，带有纸张纹理的米白色背景（#F9F7F2），使用自然的手绘线条和简笔画元素，配色柔和（暖炭灰 #3E3C38 + 珊瑚红 #FF7F7F + 鼠尾草绿 #8FA87A + 温暖黄 #FFD166 + 天空蓝 #6FBADC）。
```

**B. 布局描述**

```
[上下分层型/中心放射型/左右分栏型/流程型]布局，[描述具体的空间分配和视觉重心]
```

**C. 具体内容元素**（按视觉层次从上到下或从中心到外围描述）

```
1. 顶部：手写风格标题「[标题文字]」，[装饰元素描述]

2. 中心/上部：核心观点「[核心观点文字]」，[主视觉图形详细描述，包括简笔画人物、场景、道具等]

3. 支撑要点区域：
   - 左侧/第一个：图标 [图标描述] + 文字「[要点 1 文字]」，使用 [颜色] 强调
   - 中间/第二个：图标 [图标描述] + 文字「[要点 2 文字]」，使用 [颜色] 强调
   - 右侧/第三个：图标 [图标描述] + 文字「[要点 3 文字]」，使用 [颜色] 强调
   [继续其他要点...]

```

**D. 风格强调**

```
整体风格：手绘笔记本风格，线条自然有波浪感，避免生硬的几何形状。图标和插画使用简笔画风格，温暖亲和。信息密度高，充分利用空间，多用图形少用纯文字。
```

### 3. 调用 image-generation Skill

使用当前项目中的 `image-generation` skill 来生成图片。建议通过运行脚本的方式调用。

**工具配置**

- **脚本路径**: `skills/image-generation/scripts/nanabana_api_client.js`
- **参数说明**:
  - `--prompt`: [按上述结构编写的完整 prompt]
  - `--model`: "nano-banana-2-4k" (推荐用于高质量绘图)
  - `--aspect-ratio`: "1:1" (方形，适合社交媒体) 或 "3:4" (竖版，适合手机阅读)
  - `--image-size`: "4K" (推荐)
  - `--download`: "output/[文件名].png" (确保输出目录存在，或使用绝对路径)

**执行命令示例**:

```bash
# 示例：生成第一张图
node skills/image-generation/scripts/nanabana_api_client.js \
  --prompt "手绘风格信息图..." \
  --model "nano-banana-2-4k" \
  --aspect-ratio "3:4" \
  --image-size "4K" \
  --download "output/chart_1.png"
```

**注意**:

- 如果 `skills/image-generation/scripts/nanabana_api_client.js` 不存在，请提示用户安装 `image-generation` skill，安装路径为 `https://github.com/Time-Machine-Lab/TML-Skills/blob/master/skills/image-generation/SKILL.md`。
- 确保一次性为所有图片（3-6 张）分别调用脚本生成。
- 建议使用 `RunCommand` 工具执行命令。

### 4. 质量检查点

生成后，检查每张图是否符合要求：

- ✅ 手绘风格是否到位（温暖、亲和、自然线条）
- ✅ 内容是否完整（标题、核心观点、支撑要点、署名）
- ✅ 信息密度是否足够高（多图形、少纯文字）
- ✅ 配色是否柔和统一
- ✅ 文字是否清晰可读

### 5. 输出说明

生成完成后，向用户说明：

```
✅ 已为您生成 [X] 张手绘风格信息图！

📊 图片概览：
- 图 1：[标题] - [一句话说明核心内容]
- 图 2：[标题] - [一句话说明核心内容]
- ...

💡 使用建议：
- 这些图片适合在公众号、小红书、微博等平台分享
- 可以作为长文的配图，也可以单独发布
- 建议按顺序发布，形成完整的知识传播链条

如需调整某张图的内容或风格，请告诉我具体需求！
```

## Prompt 编写示例

**示例：图 1「远程办公新常态」**

```
手绘风格信息图，温暖亲和的视觉调性，带有纸张纹理的米白色背景（#F9F7F2），使用自然的手绘线条和简笔画元素，配色柔和（暖炭灰 #3E3C38 + 珊瑚红 #FF7F7F + 鼠尾草绿 #8FA87A + 天空蓝 #6FBADC）。

上下分层型布局，视觉重心在中心的简笔画场景，周围环绕数据卡片。

顶部：手写风格大标题「远程办公新常态」，下方有波浪形手绘下划线强调。

中心区域：一个简笔画人物坐在舒适的沙发上，膝盖上放着笔记本电脑，旁边有冒着热气的咖啡杯和小盆栽植物，背景有窗户和窗帘，营造温馨的居家办公氛围。人物表情轻松愉悦。从人物头部引出一个对话框，里面写着核心观点：「疫情后远程工作从临时方案变为长期趋势，83% 员工希望保留这一选项」。

下方三个数据卡片，从左到右排列：
- 左侧卡片：手绘风格的波浪形上升箭头（珊瑚红 #FF7F7F）+ 大数字「159%」，下方小字「全球远程工作人数 3 年增长 159%」
- 中间卡片：手绘笑脸表情（鼠尾草绿 #8FA87A）+ 数字「83%」，周围有小星星装饰，下方小字「83% 员工希望保留远程工作选项」
- 右侧卡片：手绘的房子图标和办公楼图标并列（天空蓝 #6FBADC），中间用加号连接，下方小字「混合办公成为主流模式」

整体风格：手绘笔记本风格，线条自然有波浪感，避免生硬的几何形状。图标和插画使用简笔画风格，温暖亲和。信息密度高，充分利用空间，多用图形少用纯文字。所有文字清晰可读，中文字体圆润友好。
```

## 质量标准

✅ **必须做到：**

- 使用 `image-generation` skill 生成所有图片（3-6 张）
- 每张图的 prompt 完整、详细、结构化
- 所有图片风格统一（手绘、温暖、亲和）
- 信息密度高，多用视觉元素
- 配色柔和统一

❌ **绝对禁止：**

- 遗漏任何一张图的生成
- Prompt 过于简单，导致生成效果不符合要求
- 风格不统一（有的图手绘感强，有的图太现代）
- 信息密度过低，大片留白
- 纯文字堆砌，缺乏视觉元素
