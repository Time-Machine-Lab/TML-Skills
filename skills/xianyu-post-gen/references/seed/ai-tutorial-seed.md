# 种子：AI 教程与工具资料（ai-tutorial）

这是一个关于“低价数字资料 / 教程资源 / 工具课程”的闲鱼帖子种子。

## 核心特征总结

### 1. 文案策略
- **资源清单型**：强调“包含什么、覆盖哪些模块、如何交付”，用短句列点。
- **课程/教程型**：突出“适合人群 + 解决问题 + 上手快”，减少夸大承诺。
- **超值低价型**：用“白菜价/低价整套/一口价”引导点击，但不过度营销。

### 2. 标题策略
- 结构：`[主题/工具] + [教程/资料] + [核心卖点]`
- 关键词高频：`教程`、`资料`、`PPT`、`资料合集`、`上手`、`实战`
- 字数控制：20-30 字以内，避免堆砌

### 3. 正文结构
- 商品信息（内容范围/形式/份量）
- 价格与交付（数字资料/网盘/即时发货）
- 适合人群（新手/自学/入门）
- 交易方式与边界（虚拟商品不退不换）

### 4. 可信度与合规
- 禁止“官方授权/保真/包过”等不可验证承诺
- 虚拟商品必须含“发货后不退不换”
- 避免夸大收益或不实效果

### 5. 常见标签与表情
- 标签偏好：`#闲鱼`、`#个人闲置`、`#知识付费`、`#教程`
- 表情建议：`[星星]`、`[对勾]`、`[火]`、`[包邮]`（如适用）

## 适用场景
- AIGC/ComfyUI/DeepSeek/RAG 等 AI 相关教程与资料
- 课程 PPT、教程合集、工具使用指南等数字资料类商品
