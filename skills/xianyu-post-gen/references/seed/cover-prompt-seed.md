# 闲鱼封面图提示词种子 (Cover Prompt Seed)

> **版本**: 1.1
> **更新时间**: 2026-02-26
> **用途**: 沉淀经过验证的高点击率、高清晰度闲鱼封面提示词模板。

## 1. 核心设计原则 (Core Principles)

闲鱼封面图必须在手机小屏幕信息流中瞬间抓住眼球，同时兼顾**美观、清晰、生动**：

1.  **文字即视觉 (Typography as Visual)**: 标题文字占比需超过 30%，不仅是信息，也是视觉主体的一部分。
2.  **清晰不单调 (Clear but Rich)**: 背景不必局限于纯色，推荐使用**柔和渐变、微纹理、流体、极简场景**。保证文字边缘清晰可读（必要时增加文字描边/投影）。
3.  **主体生动化 (Vivid Subject)**: 加入 3D 图标、IP 形象、悬浮物或创意合成元素，与文字形成互动，避免画面过于死板。
4.  **视觉舒适感 (Comfort)**: 配色要和谐，避免刺眼的高频噪点。

## 2. 验证通过的优质模板 (Proven Templates)

### 模板 A: 经典电商大字报 (Classic Bold) - *通用推荐*
**特点**: 醒目、直观、点击率高。通过渐变和互动元素增加丰富度。

- **Prompt 结构**:
  > 电商海报设计，扁平插画风格。[明亮色调/渐变色]背景，[装饰纹理/边框]。顶部巨大的[对比色]无衬线英文文字：[主标题]。中间是清晰的文字：[副标题]。底部是醒目的巨大文字：[核心卖点]。画面中[主体元素]与文字进行互动/穿插。色彩鲜艳，构图饱满，视觉冲击力强。

- **参考案例**:
  > 电商海报设计，扁平插画风格。明亮的柠檬黄到橙色的渐变背景，四周有断断续续的黑色涂鸦线条边框。顶部巨大的黑色无衬线英文文字：NANA_BANANA。文字稍微倾斜。中间是清晰的黑色文字：无需魔法。底部是醒目的红色对话框气泡：免费无限生成。画面右侧有一个戴着墨镜的卡通香蕉人在指着文字。色彩鲜艳，充满活力。

### 模板 B: 3D 萌趣主体 (3D Cute Object) - *质感推荐*
**特点**: 质感好、亲和力强、看着舒服。适合展示具体形象或 IP。

- **Prompt 结构**:
  > 3D C4D 电商海报，OC渲染。柔和的[奶油色/马卡龙色]背景。中央是巨大的、圆润可爱的 3D [主标题]文字。文字周围漂浮着 3D 的[装饰物品/金币/表情包]。下方悬浮文字气泡：[核心卖点]。光影柔和，材质细腻，盲盒玩具质感，画面干净整洁。

### 模板 C: 磨砂流体美学 (Glass & Fluid) - *高级感推荐*
**特点**: 看着舒服、现代、高级。适合软件、服务、美妆等。

- **Prompt 结构**:
  > 现代 UI 设计风格，磨砂玻璃拟态。背景是[颜色A]和[颜色B]的梦幻流体渐变。中央悬浮着一块半透明的磨砂玻璃卡片。卡片上写着清晰的[主标题]文字。周围漂浮着发光的[图标/几何体]。光影通透，高斯模糊，极简但有细节，视觉舒适。

### 模板 D: 超现实创意合成 (Surreal Creative) - *吸睛推荐*
**特点**: 脑洞大、有趣、生动。利用视错觉或夸张比例。

- **Prompt 结构**:
  > 超现实主义风格海报。蓝天白云/纯净空间背景。画面中央打开了一扇门/裂缝，里面流淌出[色彩/金币]。云朵/烟雾自然形成了巨大的文字：[主标题]。下方有[核心卖点]的标语牌。画面宁静而神奇，脑洞大开，细节丰富但主体突出。

## 3. 负面案例库 (Avoid List)

*   ❌ **文字过小/过细**: 必须使用 Bold/Heavy 字重，避免极简细体。
*   ❌ **背景脏乱**: 避免高频噪点（如复杂的赛博朋克故障纹理），会干扰阅读。
*   ❌ **光影过暗**: 闲鱼是亮色调 APP，暗黑风格容易被忽略（除非是专门的暗黑系产品）。
