# 闲鱼封面图提示词种子 (Cover Prompt Seed)

> **版本**: 2.1 (V1.2 + V6 融合增强版)
> **更新时间**: 2026-02-26
> **用途**: 沉淀经过验证的高点击率、高清晰度闲鱼封面提示词模板。
> **核心变更**: 恢复 V1.2 的经典模板体系，并结合 V6 (Nano-banana) 的成功经验，对不同模型的适用性进行了分级标注。

## 1. 核心设计原则 (Core Principles)

基于 `cover_v4_1_dynamic.png` (Jimeng/V4) 和 `cover_v6_1_neon_green.png` (Nano-banana/V6) 的成功经验，封面图必须遵守以下“硬性指标”：

1.  **超级大字 (Super Large Typography)**:
    *   标题文字占比必须超过 **50%**。
    *   在 Prompt 中必须使用“填满画面”、“顶天立地”、“边缘切断”等词汇描述文字大小。
    *   **拒绝小字**：在手机缩略图中，任何小于画面 1/10 的文字都是无效的。

2.  **中文优先 (Chinese First)**:
    *   针对中国市场，**除必须的英文品牌名/型号（如 nano_banana）外，其余所有副标题、卖点、标语必须使用中文**。
    *   避免使用 "AI Magic", "No Magic" 等英文表述，改为“AI魔法”、“无需魔法”。

3.  **布局绝对明确 (Explicit Layout)**:
    *   不要让 AI 自由发挥构图。明确指定：
        *   “顶部占据 2/3 区域写主标题”
        *   “底部红色横幅写卖点”
        *   “文字层级：主标题(超大) > 卖点(大) > 副标题(中)”

4.  **清晰不单调 (Clear but Vivid)**:
    *   保持高饱和度背景（黄/红/蓝/荧光绿）。
    *   加入动态元素（如骑火箭、冲破屏幕），但**绝对不能遮挡文字**。

## 2. 验证通过的优质模板 (Proven Templates)

### 模板 A: 动感大字报 (Dynamic Bold) - *Nano-banana 首选*
**特点**: 扁平矢量风格，字最大、最醒目。结合了 V1.2 的构图和 V6 的色彩策略。

- **Prompt 结构**:
  > 电商海报设计，扁平矢量插画风格。[高饱和荧光色]背景，[颜色]粗边框。
  > **排版**:
  > 1. 顶部占据画面 50% 区域，写巨大的[颜色]无衬线英文文字：[主标题(英文)]。
  > 2. 中间是清晰的[颜色]中文文字：[副标题(中文)]。
  > 3. 底部是醒目的[颜色]对话框气泡，里面写巨大的中文：[核心卖点(中文)]。
  > **装饰**: 画面右侧有一个[像素风格/扁平风格]的[动态主体元素]（如香蕉/火箭），表情兴奋。色彩极度醒目，文字边缘锐利，无阴影。

- **参考案例 (V6 验证)**:
  > 电商海报设计，扁平矢量风格。高饱和度荧光绿色背景。顶部占据画面50%区域，写巨大的黑色粗体英文：nano_banana。中间是清晰的黑色中文文字：手机直连。底部是醒目的黑色对话框，反白写巨大的中文：无需魔法。画面右侧有一个像素风格的卡通香蕉。色彩极度醒目，文字边缘锐利。

### 模板 B: 3D 堆叠大字 (3D Stacked Text) - *Jimeng 首选*
**特点**: 文字本身就是 3D 主体，占据视觉中心。质感强。
> **⚠️ 注意**: Nano-banana 模型对此风格支持较弱，容易产生模糊或伪影。建议仅在 Jimeng 或高性能模型上使用。

- **Prompt 结构**:
  > 3D C4D 电商海报，OC渲染。柔和的[颜色]背景。
  > **主体**: 画面中央堆叠着巨大的 3D 文字，占据画面 80% 空间。
  > **文字内容**: 上方是英文 [主标题]，下方是中文 [核心卖点]。
  > **装饰**: 文字周围散落着微小的 3D [装饰物]，不要抢夺视线。光影柔和，盲盒质感，文字清晰可读。

### 模板 C: 磨砂卡片 UI (Glass Card UI) - *高级感推荐*
**特点**: 信息聚焦在卡片上，强制文字排版。
> **⚠️ 注意**: 若生成文字模糊，请将“磨砂玻璃”改为“实色卡片”或“扁平化UI”。

- **Prompt 结构**:
  > 现代 UI 设计风格。背景是[颜色]流体渐变。
  > **主体**: 中央悬浮一张巨大的磨砂玻璃卡片（或白色不透明卡片），占据画面 70%。
  > **卡片内容**: 卡片内部排版严谨。上方是黑色的粗体英文：[主标题]。下方是醒目的中文按钮：[核心卖点]。
  > **细节**: 光影通透，文字边缘锐利，没有任何英文废话。

### 模板 D: 孟菲斯波普 (Memphis Pop) - *V6 验证推荐*
**特点**: 设计感强，色彩跳跃，年轻化。V6 验证效果极佳。

- **Prompt 结构**:
  > 孟菲斯设计风格海报，波普艺术。高饱和度[颜色]背景，点缀白色的波点图案。
  > **文字**: 画面正中央横跨着巨大的[颜色]几何色块，上面写着巨大的粗体中文/英文：[主标题]。
  > **中文强调**: 上下角落分别用醒目的标签纸风格写着中文：[卖点1] 和 [卖点2]。
  > **风格**: 极简、高对比、大字号、二维设计、无阴影。

## 3. 模型适配指南 (Model Compatibility)

为了保证出图率，请根据使用的模型选择合适的风格关键词：

| 特性 | **Nano-banana (当前主力)** | **Jimeng / Midjourney** |
| :--- | :--- | :--- |
| **首选风格** | **Flat Vector (扁平矢量)**, Pop Art (波普), Minimalist (极简) | 3D C4D, OC Render, Clay style (粘土风) |
| **推荐配色** | **Neon (荧光色)**, Solid Colors (纯色), High Contrast (高反差) | Gradient (渐变), Soft lighting (柔光), Pastel (粉彩) |
| **避坑指南** | ❌ 避免复杂的“光影”、“渲染”、“玻璃质感”词汇，会导致文字扭曲。 | ✅ 可以尝试复杂的材质和光影。 |
| **最佳模板** | **模板 A (动感大字报), 模板 D (孟菲斯)** | **模板 B (3D 堆叠), 模板 C (磨砂卡片)** |

## 4. 负面案例库 (Avoid List)

*   ❌ **英文副标题**: 禁止出现 "Pro Version", "High Quality" 等，全部换成 "专业版", "高清画质"。
*   ❌ **文字悬浮在云端/烟雾中**: 这种效果通常导致文字扭曲或太小，不易阅读。
*   ❌ **文字占比 < 30%**: 这种图在闲鱼必死。
*   ❌ **复杂背景纹理**: 干扰文字阅读的纹理一律去掉（特别是在 Nano-banana 上，纹理会变成噪点）。
