# 闲鱼封面图提示词种子 (Cover Prompt Seed)

> **版本**: 1.2
> **更新时间**: 2026-02-26
> **用途**: 沉淀经过验证的高点击率、高清晰度闲鱼封面提示词模板。

## 1. 核心设计原则 (Core Principles)

基于 `cover_v4_1_dynamic.png` 的成功经验，封面图必须遵守以下“硬性指标”：

1.  **超级大字 (Super Large Typography)**:
    *   标题文字占比必须超过 **50%**。
    *   在 Prompt 中必须使用“填满画面”、“顶天立地”、“边缘切断”等词汇描述文字大小。
    *   **拒绝小字**：在手机缩略图中，任何小于画面 1/10 的文字都是无效的。

2.  **中文优先 (Chinese First)**:
    *   针对中国市场，**除必须的英文品牌名/型号（如 nano_banana）外，其余所有副标题、卖点、标语必须使用中文**。
    *   避免使用 "AI Magic", "No Magic" 等英文表述，改为“AI魔法”、“无需魔法”。

3.  **布局绝对明确 (Explicit Layout)**:
    *   不要让 AI 自由发挥构图。明确指定：
        *   “顶部占据 2/3 区域写主标题”
        *   “底部红色横幅写卖点”
        *   “文字层级：主标题(超大) > 卖点(大) > 副标题(中)”

4.  **清晰不单调 (Clear but Vivid)**:
    *   保持高饱和度背景（黄/红/蓝）。
    *   加入动态元素（如骑火箭、冲破屏幕），但**绝对不能遮挡文字**。

## 2. 验证通过的优质模板 (Proven Templates)

### 模板 A: 动感大字报 (Dynamic Bold) - *强烈推荐*
**特点**: 字最大、最醒目、有动态感。复刻自 `cover_v4_1`。

- **Prompt 结构**:
  > 电商海报设计，扁平插画风格。[高饱和色]背景，[颜色]粗边框。
  > **排版**:
  > 1. 顶部占据画面 50% 区域，写巨大的[颜色]无衬线英文文字：[主标题(英文)]。
  > 2. 中间是清晰的[颜色]中文文字：[副标题(中文)]。
  > 3. 底部是醒目的[颜色]对话框气泡，里面写巨大的中文：[核心卖点(中文)]。
  > **装饰**: 画面右侧有一个[动态主体元素]（如骑火箭/奔跑/冲刺），表情兴奋。色彩鲜艳，文字填满画面。

- **参考案例**:
  > 电商海报设计，扁平插画风格。明亮的黄色背景，黑色粗边框。顶部巨大的黑色无衬线英文文字：nano_banana。中间是清晰的黑色文字：无需魔法_国内直连。底部是醒目的红色对话框气泡：免费+无限生成。画面右侧有一个骑着火箭飞行的卡通香蕉，表情兴奋。色彩鲜艳，充满活力，构图饱满。

### 模板 B: 3D 堆叠大字 (3D Stacked Text) - *质感推荐*
**特点**: 文字本身就是 3D 主体，占据视觉中心。

- **Prompt 结构**:
  > 3D C4D 电商海报，OC渲染。柔和的[颜色]背景。
  > **主体**: 画面中央堆叠着巨大的 3D 文字，占据画面 80% 空间。
  > **文字内容**: 上方是英文 [主标题]，下方是中文 [核心卖点]。
  > **装饰**: 文字周围散落着微小的 3D [装饰物]，不要抢夺视线。光影柔和，盲盒质感，文字清晰可读。

### 模板 C: 磨砂卡片 UI (Glass Card UI) - *高级感推荐*
**特点**: 信息聚焦在卡片上，强制文字排版。

- **Prompt 结构**:
  > 现代 UI 设计风格。背景是[颜色]流体渐变。
  > **主体**: 中央悬浮一张巨大的磨砂玻璃卡片，占据画面 70%。
  > **卡片内容**: 卡片内部排版严谨。上方是黑色的粗体英文：[主标题]。下方是醒目的中文按钮：[核心卖点]。
  > **细节**: 光影通透，文字边缘锐利，没有任何英文废话。

### 模板 D: 孟菲斯几何中文版 (Memphis Chinese)
**特点**: 设计感强，但必须保证文字够大。

- **Prompt 结构**:
  > 孟菲斯风格海报。[颜色]背景，点缀几何图形。
  > **文字**: 画面正中央横跨着巨大的黑色粗体文字：[主标题]。文字背景有错位色块。
  > **中文强调**: 上下角落分别用醒目的标签纸风格写着中文：[卖点1] 和 [卖点2]。
  > **风格**: 极简、高对比、大字号。

## 3. 负面案例库 (Avoid List)

*   ❌ **英文副标题**: 禁止出现 "Pro Version", "High Quality" 等，全部换成 "专业版", "高清画质"。
*   ❌ **文字悬浮在云端/烟雾中**: 这种效果通常导致文字扭曲或太小，不易阅读。
*   ❌ **文字占比 < 30%**: 这种图在闲鱼必死。
*   ❌ **复杂背景纹理**: 干扰文字阅读的纹理一律去掉。
