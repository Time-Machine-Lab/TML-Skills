# 小红书治愈系壁纸热门帖子特征 (Healing Wallpaper Hot Posts)

## 总结：给 Agent 的生成建议

基于以上真实案例，生成“治愈系/情绪价值”壁纸内容时应遵循：

1. **标题策略**：
   * **场景+情感**：`城市/季节 | 你的心事`
   * **热词+利益点**：`多巴胺/停止焦虑 | 治愈/暴富`
   * **对话式**：`宝！你要记住！` / `你的城市下雪了吗？`
2. **文案调性**：
   * **温柔陪伴型**：使用“我们”、“你”、“值得”等词汇，提供拥抱感。
   * **可爱拟人型**：赋予物体生命（小火山、云朵），使用软萌语气（~, 呀, 呢）。
   * **清醒大女主型**：简短有力，拒绝内耗。
3. **视觉描述**：
   * 强调“插画”、“手绘”、“暖色调”、“风景”、“光影”。
4. **标签组合**：
   * 基础词：#壁纸 #手机壁纸 #锁屏
   * 情绪词：#治愈 #温暖 #少女心 #正能量 #解压
   * 场景/风格词：#插画 #风景 #多巴胺 #ins风
