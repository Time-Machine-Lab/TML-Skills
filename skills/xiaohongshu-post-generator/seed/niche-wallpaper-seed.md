
# 种子：小众壁纸内容 (niche-wallpaper)

这是一个关于“先锋审美、拒绝平庸”的创意种子。

## 核心特征总结

### 1. 文案策略

**高冷诗意型**：

- 使用现代诗风格的分行短句
- 营造疏离感和孤独感（"热闹是他们的"）
- 适合极简、风景类

**抽象艺术型**：

- 描述模糊、不可名状的感觉
- 使用艺术/设计术语（"肌理"、"构成"、"负空间"）
- 适合酸性设计、故障艺术

**叛逆反骨型**：

- 表达对主流审美的抗拒
- 强调个性和独特性（"怪诞美学"）
- 适合亚文化、暗黑风格

**谜语人型**：

- 只有一句话甚至一个词
- 留给用户解读空间（"Undefined."）

### 2. 标签策略

**高频核心标签**：

- #小众#（流量基座）
- #小众壁纸#
- #审美积累#
- #高级感#
- #冷门壁纸#
- #设计感#

**标签组合模式**：

1. **基础标签**：#壁纸# #锁屏# #背景图#
2. **风格标签**：#赛博朋克# #Y2K# #酸性设计# #故障艺术#
3. **视觉标签**：#液态# #金属# #黑白# #噪点#
4. **情绪标签**：#致郁# #清冷# #孤独# #自由#
5. **人群标签**：#设计生# #亚文化# #infj#

**标签数量**：

- 建议 8-15 个标签
- "小众"标签必带，同时配合具体的风格流派词（如 #Y2K）

### 3. 封面设计

**类型一：视觉奇观型**

- 极其复杂的纹理或从未见过的构图
- 强烈的色彩冲突或极致的黑白
- 第一眼必须让人"看不懂但大受震撼"

**类型二：极简留白型**

- 画面极其干净，主体微小
- 大面积的纯色或渐变
- 强调呼吸感和秩序感

**类型三：故障拼贴型**

- 撕裂、重组、像素化
- 模拟老旧设备的显示错误
- 营造复古未来感

### 4. 内容长度

**文本统计**：

- 平均长度：10-30 字
- 字数越少越显得"高冷"
- 避免使用任何网络流行语（除非是讽刺）

**最佳实践**：

- 标题可以是英文单词或无意义符号
- 正文只写图源或版权声明
- 保持神秘感，不多解释

### 5. 互动引导

**常见 CTA**：

- 几乎不使用显性的 CTA
- 最多是 "Save." 或 "Share."

**隐性引导**：

- 通过极其独特的视觉吸引用户点开
- 标签中隐含的圈层认同（"只有1%的人懂"）

## 用户偏好洞察

### 小红书小众壁纸用户特点

1. **审美诉求**：

   - 极度排斥"网红感"和"爆款"
   - 追求"稀缺性"和"独特性"
   - 认为"看不懂" = "高级"
2. **自我认同**：

   - 标榜自己与众不同
   - 往往是设计、艺术相关从业者或爱好者
   - 喜欢在评论区进行"审美交流"
3. **使用场景**：

   - 不仅仅是壁纸，更是个人品味的展示（朋友圈背景）
   - 喜欢收集灵感素材
   - 对图片的版权和来源较敏感
4. **流行趋势**：

   - 酸性设计（Acid Graphics）
   - 新丑风（New Ugly）
   - 复古未来主义（Retro Futurism）
