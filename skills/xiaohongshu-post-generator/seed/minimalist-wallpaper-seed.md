# 种子：极简主义与功能美学壁纸 (minimalist-wallpaper)

这是一个关于“克制、实用、信仰感”的创意种子。

## 核心特征总结

### 1. 文案策略

**功能实用型（痛点打击）**：

- **适配性**：强调与系统UI的完美融合（"完美适配iOS时间组件"、"灵动岛壁纸"、"不挡时间"）。
- **硬件友好**：强调对设备的优化（"OLED屏福音"、"纯黑省电"、"夜间护眼"）。
- **场景化**：直接指出使用场景（"开会敢用的壁纸"、"适合考研党的专注壁纸"）。

**信仰美学型（情绪升华）**：

- **赋予意义**：将简单的行为仪式化（"每次解锁，都是一次朝圣"、"自律的开始"）。
- **克制冷淡**：使用简短、有力、带有疏离感的句子（"保持冷静"、"Less is More"）。
- **内心投射**：关联用户内心状态（"平息浮躁"、"在喧嚣中独处"）。

**反差修辞型**：

- "五彩斑斓的黑"
- "最吵闹的安静"
- 利用视觉与心理的反差制造记忆点。

### 2. 标签策略

**高频核心标签**：

- #极简壁纸#（流量基座）
- #锁屏美学#
- #黑色壁纸#
- #手机壁纸#
- #高级感#

**标签组合模式**：

1. **基础标签**：#壁纸 #锁屏 #背景图
2. **功能标签**：#省电壁纸 #护眼模式 #iOS16 #灵动岛壁纸 #全面屏壁纸
3. **风格标签**：#冷淡风 #暗黑系 #黑白 #像素风 #ins风
4. **情绪/属性标签**：#信仰 #自律 #清醒 #专注 #打工人
5. **色系标签**：#纯黑 #黑白灰 #莫兰迪色

**标签数量**：

- 建议 10-15 个标签。
- **必带组合**：场景词（锁屏）+ 功能词（适配/省电）+ 风格词（极简/信仰）。

### 3. 封面设计

**类型一：实机效果展示型（Mockup）**

- **直接上机**：展示壁纸在手机锁屏界面（包含时间、日期、小组件）的实际效果。
- **痛点证明**：直观展示“不挡时间”、“与小组件排版和谐”。
- **增加信赖**：用户能立刻想象用在自己手机上的样子。

**类型二：极致留白/沉底构图**

- **大面积纯色**：通常是纯黑或纯白，占比 70% 以上。
- **主体微小**：主体（如文字、小图标、像素画）位于屏幕底部或正中央极小区域。
- **呼吸感**：强调画面的空旷和秩序。

**类型三：质感特写型**

- 聚焦于材质细节（如水泥纹理、纸张褶皱、流体光泽）。
- 去除具体形象，只保留氛围。

### 4. 内容长度

**文本统计**：

- 平均长度：30-60 字。
- 分行短句为主，类似诗歌排版。
- 即使是功能性介绍，也要写得有格调。

**最佳实践**：

- **标题**：痛点/价值 + 风格定义（如“锁屏美学｜...”、“省电又高级｜...”）。
- **正文**：
    - 第一段：一句话定义这张图的气质（“你的手机需要一点信仰”）。
    - 第二段：列举功能卖点（“适配iOS”、“省电”）。
    - 第三段：情绪升华（“平息浮躁”）。
- **排版**：善用空格和换行，保持视觉上的稀疏感。

### 5. 互动引导

**常见 CTA**：

- "换上截屏给我看"（极其有效的互动，引发评论区晒图）。
- "强迫症舒适区，进"
- "蹲一个同款"

**隐性引导**：

- 利用“系列感”引导关注（“下期更白色系”）。
- 利用“适配机型”引发讨论（“14Pro可以用吗？”）。
