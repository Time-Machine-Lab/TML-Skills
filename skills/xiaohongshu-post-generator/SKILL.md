# 小红书帖子内容孵化器

该 Skill 旨在帮助团队围绕一个核心创意（种子），结合平台热点数据，批量繁衍出多个具有爆款潜质的小红书帖子。

## 核心概念

### 1. 种子 (Seed)

- **定义**：一个特定内容领域的“浓缩精华”和“生长基因”。它不仅仅是一个主题，更包含该领域的视觉符号、情绪触发点、反逻辑（Anti-Logic）和可拓展场景。
- **作用**：作为内容生成的根因，所有繁衍内容（叶子）都必须围绕此核心展开，但必须具备**高随机性**和**强变异性**。
- **存储位置**：`seed/*.md`
- **示例**：`seed/niche-wallpaper-seed.md` (小众壁纸内容)

### 2. 素材库 (Knowledge Base)

- **种子方向素材库**：
  - **定义**：该种子方向所需的特定素材、风格指南或垂直领域知识。
  - **存储位置**：`document/seed/<种子文件名>/` 文件夹下的所有 markdown 文件。
  - **示例**：如果种子是 `niche-wallpaper-seed`，则素材在 `document/seed/niche-wallpaper-seed/*.md`。
- **热门数据素材库**：
  - **定义**：当前平台该方向的所有热门文章素材，包含实时热点、爆款因子、用户偏好分析。
  - **存储位置**：`document/hot/*.md`

## 功能一：种子内容生成

当用户指令如：“基于 **[种子名称]** 风格生成文案”时，请严格遵循以下步骤：

### 第一步：种子解析 (Seed Analysis)

1. 定位并读取 **种子文档**：`seed/[种子名称].md`。
2. 深度理解种子的 **视觉符号 (Visual Symbols)**、**情绪钩子 (Emotional Hooks)** 和 **反常识逻辑 (Anti-Logic)**。

### 第二步：素材融合 (Knowledge Fusion)

1. **加载种子素材**：
   - 找到与种子同名的素材文件夹：`document/seed/[种子名称]/`。
   - 读取该文件夹下的所有 `.md` 文件。
   - *目的*：确保生成内容符合该垂直领域的专业性和特定风格。
2. **加载热门素材**：读取 `document/hot/` 下的相关热门文档。
   - *目的*：提取“实时热点”、“爆款因子”（如点击诱饵标题、高赞互动话术、流行梗）。

### 第三步：随机化策略注入 (Randomization Strategy)

为了避免内容平庸，必须在生成前注入以下随机策略之一：

- **混搭 (Remix)**：强行组合两个看似不相关的种子元素（如：将“赛博朋克”与“田园风”结合）。
- **反差 (Contrast)**：使用与画面极度不符的文案（如：画面很暗黑，文案很治愈）。
- **具体化 (Specification)**：拒绝宏大叙事，聚焦极其微小的细节（如：不写“夏天”，写“冒着气泡的冰可乐”）。
- **人味儿 (Human Touch)**：模拟真人的口语习惯，甚至包含微小的瑕疵（如：倒装句、语气词、无标点）。

### 第四步：内容繁衍 (Content Generation)

基于上述分析，生成 3-5 个不同方向的帖子内容。每个帖子必须包含：

1. **抓人眼球的标题 (Hook Title)**：
   - 必须在前 5 个字内抓住用户。
   - 使用 **[种子]** 中的 **反逻辑** 或 **视觉符号**。
   - 拒绝平铺直叙，必须有情绪或信息增量。
2. **沉浸式正文 (Immersive Content)**：
   - **风格适配**：根据种子的目标受众，从以下风格中选择最贴切的一种：
     - **清冷诗意风**（适合小众/艺术类）：短句换行，大量留白，使用 "..."、"。" 或无标点。Emoji 克制（🖤、🌫️、🥀）。
     - **元气碎碎念风**（适合治愈/生活类）：口语化，像朋友聊天，使用语气词（呀、呢、~）。Emoji 丰富且可爱（✨、🌈、🧸、🌷）。
     - **极简高冷风**（适合潮酷/设计类）：字数极少，可能只有一句英文或几个关键词。Emoji 使用几何图形或冷门符号（▪️、©️、®️、💿）。
     - **情绪发泄风**（适合共鸣/吐槽类）：感叹号多，情绪强烈，直击痛点。Emoji 使用表情夸张的（😭、🙏、💥）。
   - **排版美学**：
     - 善用空行进行分段，避免大段文字。
     - 使用特殊符号作为列表项（如 ✦、•、- ）。
     - 关键信息加粗或使用【】强调。
   - **场景化**：不要说“这个壁纸好看”，要说“深夜两点换上这张图，我不想睡了”。
   - **情绪共鸣**：提供情绪价值（抚慰、宣泄、激励、优越感）。
3. **精准流量标签 (Traffic Tags)**：
   - 组合：大盘词（流量基座）+ 垂直词（精准人群）+ 长尾词（搜索优化）。

## 功能二：种子反向提取 (Seed Extraction)

当用户提供一批素材，并要求“提取种子因子”或“生成种子文档”时：

### 1. 输入处理

- 用户可能会提供一段文本、几个 URL，或者指定一个现有的素材文件夹路径（如 `document/seed/[new-topic]/`）。

### 2. 因子分析 (Factor Analysis)

Agent 需要深入阅读提供的素材，提取以下 **种子因子 (Seed Factors)**：

- **核心定义 (Core Definition)**：这到底是什么？（一句话定义主题）
- **目标受众 (Target Audience)**：谁在看？（画像：年龄、兴趣、痛点）
- **核心特征总结 (Core Features)**：
  - **文案策略**：总结 3-4 种常见的文案风格（如诗意型、极简型）。
  - **标签策略**：提取核心标签及组合模式。
  - **封面设计**：分析封面图的视觉类型和构图特征。
  - **互动引导**：识别常见的 CTA（行动号召）话术。
- **用户偏好洞察 (User Insights)**：分析该领域用户的审美导向、情感投射及消费习惯。
- **风格流派 (Style Genres)**：有哪些典型的视觉/内容风格？（枚举 3-5 个流派）
- **视觉符号 (Visual Symbols)**：该领域常出现的具体物体（如：月亮、霓虹灯、猫、咖啡）。
- **情绪钩子 (Emotional Hooks)**：用户看这个内容是为了获得什么情绪？（如：治愈、爽感、优越感）。
- **反常识逻辑 (Anti-Logic)**：该领域有哪些打破常规的认知？（如：越丑越火、越怪越高级）。
- **语言调性 (Tone of Voice)**：文案的语气是怎样的？（如：高冷、逗比、治愈、专业）。
- **高频关键词 (Keywords)**：出现频率最高的 5-10 个词。

### 3. 种子生成 (Seed Generation)

基于提取的因子，按照标准格式生成一个新的种子文档，并保存到 `seed/[new-topic].md`。

**标准种子模板**：

```markdown
# 种子：[主题名称] ([topic-code])

这是一个关于 [核心定义] 的创意种子。

## 核心特征总结

### 1. 文案策略
- **策略类型A**：
  - 特征描述...
  - 适用场景...
- **策略类型B**：
  - 特征描述...

### 2. 标签策略
- **高频核心标签**：#标签1 #标签2 ...
- **标签组合模式**：
  1. **基础标签**：...
  2. **风格标签**：...
  3. **场景标签**：...
- **标签数量建议**：...

### 3. 封面设计
- **类型一**：描述...
- **类型二**：描述...

### 4. 内容长度
- **文本统计**：平均字数、标点偏好...
- **最佳实践**：...

### 5. 互动引导
- **常见 CTA**：...
- **隐性引导**：...

---

## 用户偏好洞察

### [主题]用户特点
1. **审美导向**：...
2. **情感投射**：...
3. **消费习惯**：...
4. **热门趋势**：...

```
