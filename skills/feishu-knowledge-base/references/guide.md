## 概念速览
- 知识库（Wiki 空间）：面向组织的知识管理系统，每个知识空间有唯一 space_id。
- 文档（Docx 文档）：云文档中的一篇在线文档，具有唯一 document_id；可存储于云盘，也可挂载为知识库节点。
- 块（Block）：文档中的最小构建单元，每个块有唯一 block_id。
- 层级与关联：
  - 知识库 ← 包含 → 文档 ← 由 → 块 组成。
  - 文档挂载为节点：通过获取知识空间节点信息得到 obj_token 与 obj_type；当 obj_type 为 docx 时，obj_token 即 document_id。
  - 文档由块构成：根块为页面块 Page Block，其 block_id 与 document_id 相同；其他块与页面块构成父子关系。
- 特性与示例：
  - 目录本身也是文档，可存内容并承载子文档，支持多级嵌套。
  - 示例：test 知识库 → test 文档 → test 介绍 文档。
- 常见块类型：文本类（页面/文本/标题/列表）、数据类（多维表格/电子表格/思维笔记）、媒体类（图片/文件/内嵌网页）、协作类（会话卡片）。
- 实操对应：结构使用 wiki-space-node 系列，内容读写使用 docx 系列，创建子文档需指定父节点 node_token/ID。
